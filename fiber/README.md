# Products API

## Описание
RESTful API для управления продуктами с полным набором CRUD операций.


## Эндпоинты

### Получить все продукты
- **Метод:** `GET`
- **URL:** `/products`
- **Описание:** Получить список всех продуктов
- **Ответ:** `200 OK` - Массив объектов продуктов

### Получить продукт по ID
- **Метод:** `GET`
- **URL:** `/products/:id`
- **Параметры:**
  - `id` (в пути) - ID продукта
- **Ответы:**
  - `200 OK` - Объект продукта
  - `404 Not Found` - Продукт не найден

### Создать продукт
- **Метод:** `POST`
- **URL:** `/products`
- **Тело:** Объект продукта (JSON)
- **Ответы:**
  - `201 Created` - Созданный продукт
  - `400 Bad Request` - Неверные данные

### Обновить продукт
- **Метод:** `PUT`
- **URL:** `/products/:id`
- **Параметры:**
  - `id` (в пути) - ID продукта для обновления
- **Тело:** Обновленный объект продукта (JSON)
- **Ответы:**
  - `200 OK` - Обновленный продукт
  - `404 Not Found` - Продукт не найден
  - `400 Bad Request` - Неверные данные

### Удалить продукт
- **Метод:** `DELETE`
- **URL:** `/products/:id`
- **Параметры:**
  - `id` (в пути) - ID продукта для удаления
- **Ответы:**
  - `204 No Content` - Успешно удален
  - `404 Not Found` - Продукт не найден

## Примеры использования

```curl -X GET http://localhost:3000/api/v1/products/123```

```curl -X POST http://localhost:3000/api/v1/products \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Новый продукт",
    "price": 99.99,
    "description": "Описание продукта"
  }'```
  
 ``` curl -X PUT http://localhost:3000/api/v1/products/123 \

  }'```
